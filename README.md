# IP Normalization

A quick script to bulk transform IP addresses into CIDR range. prints to `STDOUT`

usage:
   `./ipnorm.py IPFILE`

> notes : (1) prints to stdout. Redirect to file.  (2). makesure `netaddr` installed.
